内容类型

Netflix 作为全球领先的流媒体平台，截至2021年中期，其平台拥有超过2亿订阅用户，可访问超过8,000部电影和电视节目。本数据集是Netflix平台上所有内容的完整编目。

字段名 类型 说明 示例值
show_id string 每个标题的唯一标识符 s1, s2...
type string 标题类别（电影/电视节目） Movie, TV Show
title string 电影或电视节目的名称 Dunkirk
director string 导演（电影可能有，剧集通常无） Christopher Nolan
cast string 主要演员列表（逗号分隔） Tom Hardy, Cillian Murphy
country string 制作国家（逗号分隔） United States, United Kingdom
date_added string 添加至Netflix的日期 September 17, 2021
release_year int 原始发布年份 2017
rating string 年龄评级 TV-MA, PG-13
duration string 时长（分钟或季数） 116 min, 4 Seasons
listed_in string 所属流派（逗号分隔） Action, Thriller
description string 简短概述 A group of soldiers...

数据分析目标 REQUIRED

根据考核要求，本次数据分析需实现以下六大核心目标，每个目标都应有明确的可交付成果：

01
描述性统计分析
计算各字段的基本统计量，如均值、中位数、标准差等，揭示数据的基本特征
02
类型分布分析
分析电影与电视节目的比例、各类型内容的数量分布
03
时间趋势分析
探索内容发布时间趋势，分析Netflix平台内容增长的规律
04
地理分布分析
分析内容制作的国家/地区分布，识别主要内容产出地
05
相关性分析
探索不同变量之间的相关性，如评级与时长、年份与类型等
06
机器学习建模
应用分类、聚类、回归等算法，挖掘数据中的深层规律
