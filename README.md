# Netflix 数据分析可视化看板

## 📊 项目简介
本项目基于 Netflix 公开数据集，通过 Python 进行数据清洗、分析和可视化，最终构建了一个交互式 Web 可视化看板。项目旨在探索 Netflix 内容库的分布特征、发展趋势和内容偏好，为内容策略提供数据支持。

## 🎯 分析目标
- 探索电影与电视节目的比例分布
- 分析内容发布的时间趋势
- 研究不同国家/地区的内容产出情况
- 识别热门内容流派和评级分布
- 发现内容时长和季节数的规律
- 提供基于数据的业务建议

## 📊 数据集说明
- **来源**: Netflix 公开数据集
- **记录数**: 8,807 条
- **字段数**: 23 个
- **主要字段**: 
  - `type`: 内容类型（Movie/TV Show）
  - `title`: 标题
  - `country`: 制片国家
  - `release_year`: 发布年份
  - `rating`: 内容评级
  - `duration`: 时长
  - `listed_in`: 流派分类
  - `date_added`: 上架日期

## 🔍 核心发现
1. **内容类型**: 电影占比 69.6%，电视节目占比 30.4%
2. **增长趋势**: 2019 年达到发布峰值（2,016 部）
3. **地域分布**: 美国为最大制片国（3,690 部）
4. **受众定位**: TV-MA（成人级）内容最多
5. **热门流派**: International Movies 最受欢迎
6. **内容时长**: 电影时长集中在 90-120 分钟

## 💡 业务建议
1. **内容策略**: 继续加大国际电影投入，满足全球化需求
2. **区域拓展**: 关注印度、英国等高潜力市场
3. **分级优化**: 针对 TV-MA 受众开发更多优质内容
4. **时段规划**: 在 90-120 分钟时长区间增加内容储备
