# Netflix数据分析案例报告 - 前端展示页面规划

## 一、项目概述

### 1.1 项目背景
- **案例报告主题**: Netflix影视作品数据分析
- **展示形式**: 多页面前端网站
- **技术栈**: HTML + CSS + JavaScript（纯前端，无需构建）

### 1.2 设计风格
- **主色调**: Netflix红 `#E50914` + 深黑 `#141414`
- **字体**: 思源黑体(中文) + Montserrat(英文)
- **风格**: 现代简约、数据驱动，品牌一致性

### 1.3 小组成员分工

| 成员 | 学号 | 主要任务 |
|------|------|----------|
| 成员1 | - | 数据预处理与清洗（缺失值处理、异常值修复、特征提取） |
| 成员2 | - | 随机森林算法实现、可视化图表生成、前端页面制作、报告撰写 |

---

## 二、页面规划

### 页面结构（共5个页面）

| 页面 | 文件名 | 主要内容 |
|------|--------|----------|
| 首页 | `index.html` | 项目概览、成员分工、导航入口 |
| 项目需求分析 | `analysis.html` | 数据集背景、变量说明、分析目标 |
| 数据处理代码 | `code.html` | 数据预处理、随机森林算法代码 |
| 可视化结果 | `visualization.html` | 10张图表展示 |
| 案例分析结论 | `conclusion.html` | 主要发现、结论与建议 |

## 五、开发顺序

1. **CSS样式文件** - 定义全局样式、组件
2. **index.html** - 首页（包含导航和页脚）
3. **analysis.html** - 项目需求分析
4. **code.html** - 数据处理代码
5. **visualization.html** - 可视化结果
6. **conclusion.html** - 案例分析结论
7. **main.js** - 交互脚本

## 六、关键实现点

| 功能 | 实现方式 |
|------|----------|
| 导航高亮 | JavaScript + CSS |
| 代码高亮 | Prism.js CDN |
| 图表展示 | CSS Grid + 灯箱效果 |
| 响应式 | CSS Media Queries |
| 动画效果 | CSS Transitions |
| 懒加载 | loading="lazy" |
